# 결혼이민자의 국적취득

## 📌 간이귀화란?

간이귀화는 한국과 특별한 관계가 있는 외국인을 위해 만든 제도입니다.

일반귀화보다 조건이 조금 쉬운 귀화 방법입니다.

***

## 📌 간이귀화 종류

* 혼인유지\
  → 한국인 배우자와 결혼 상태를 유지하는 경우
* 혼인단절\
  → 이혼, 사망 등으로 혼인이 끝난 경우
* 3년 이상 거주\
  → 부모, 출생 등 한국과 관계가 있는 경우

***

## 📌 공통 요건

다음 조건을 모두 만족해야 합니다.

* 대한민국 민법상 성년일 것
* 법을 잘 지키고 품행이 단정할 것
* 생활할 수 있는 경제 능력이 있을 것
* 한국어를 이해하고 한국 문화를 기본적으로 이해할 것
* 국가 안전과 질서를 해치지 않을 것

***

## ⚠️ 중요 안내

* 외국어 서류는 반드시 한국어로 번역해야 합니다
* 심사 중 추가 서류를 요구할 수 있습니다
* 거짓 서류 제출 시 허가가 거절될 수 있습니다
